# 環境設計(最新は2023/06/18時点)

## サーバー環境
- OS: Ubuntu Server 22.04.2 LTS(略: Jammy Jellyfish)
- サーバー: Docker環境(Docker Compose)
  - Webサーバコンテナ: nginx 1.24.0(最新安定版); nginx:1.24.0-bullseye
